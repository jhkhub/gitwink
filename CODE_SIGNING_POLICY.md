# Code signing policy

gitwink uses free code signing provided by [SignPath.io](https://about.signpath.io/),
with a free code signing certificate by the [SignPath Foundation](https://signpath.org/).

## Team roles

gitwink is a single-maintainer open-source project.

- **Committers and reviewers:** [@var-gg](https://github.com/var-gg)
- **Approvers:** [@var-gg](https://github.com/var-gg)

## Privacy policy

gitwink does not collect or transmit personal information. Its only
networked activity is an optional update check against GitHub Releases,
performed on the user's behalf. The full privacy policy is in
[PRIVACY.md](PRIVACY.md).
